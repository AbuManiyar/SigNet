# Signature verification system using Siamese networks
![Siamese Network](https://github.com/AbuManiyar/SigNet/blob/master/image.png)
## Objectives

1. **Develop an automated offline signature verification system** using deep learning techniques to distinguish between genuine and forged signatures.

2. **Implement a Siamese neural network architecture** that learns similarity between pairs of signature images using metric learning.

3. **Improve the baseline model by designing a SigNet-2 inspired architecture**, incorporating additional layers to learn more discriminative signature embeddings.

4. **Evaluate the model on a benchmark signature dataset** using writer-independent testing and performance metrics such as accuracy, precision, recall, and F1-score.

## Dataset
> Refered Kaggle [Dataset](https://www.kaggle.com/datasets/akashgundu/signature-verification-dataset)
